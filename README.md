# FungalAssemblerPipeline
A pipeline dedicated to the assembly of fungal genomes with ONT reads, and polishing with .

## Dependencies

  * Assemblers (At least one):
    * Miniasm
    * Canu
    * Spades
  * Polishing (At least one):
    * Racon
    * Pilon
  * Metrics (At least one):
    * BUSCO
      * Augustus
    * QUAST
  * Core tools
    * Samtools
    * Pandas
